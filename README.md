# Evans Mutisya - Cybersecurity Portfolio

Professional portfolio website for Evans Mutisya, IT Systems and Cybersecurity Specialist.

## 🚀 Features

- Modern, responsive design with cybersecurity theme
- Matrix-style animated background
- Skills matrix visualization
- Project showcase with filtering
- Professional experience timeline
- Certifications and education display
- Contact form with email integration
- Downloadable resume
- SEO optimized

## 🛠️ Technologies

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
